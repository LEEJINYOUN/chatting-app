# 채팅 앱

<h2> 📃 개요 </h2>

- <h3>로그인한 후에 실시간으로 채팅을 보낼수 있는 서비스</h3>
  <br/><br/>

<h2> 🛠 사용 기술 </h2>

- <h3>React</h3>
- <h3>Firebase</h3>
  <br/><br/>

<h2> ⚙️ 구현 기능 </h2>

- <h3>실시간 채팅 보내기 기능</h3>
- <h3>로그아웃 기능</h3>
